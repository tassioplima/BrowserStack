# BrowserStack
Automation with Gradle + Java on Browser Stack remote mobile

![Appium Execution](https://github.com/tassioplima/BrowserStack/workflows/Gradle/badge.svg)


Configuration: 

JDK 8 or 11
Android SDK updated
Android Emulator
Appium Server
Browser Stack Account and Keys

Running:

```
./gradlew test --tests Tests
```
